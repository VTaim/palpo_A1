This directory includes a set of simple semantic modeling examples a la
Python & RDFLib (+rdflib-sqlalchemy with persistent rdf stores).

A recommended order and grouping of the files is the following:

readme.txt - this file

calvin.ttl - sample data	

read-add-write.py - basic I/O

navigate.py	- basic management

merge.py - merging RDF graphs		
part1.ttl	- sample data for merging
part2.ttl - "

persistence-create.py - creating a persistent rdf store (sqlite)
persistence-add.py - adding statements to an existing store
		
sparql-construct.py - simple sparql examples
sparql-select.py - "
